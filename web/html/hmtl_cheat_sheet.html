Actividad: HTML Cheat Sheet

Tag: a

Anchor se utiliza para dar propiedades de hipervínculos a un texto. Dicho texto se podrea clickear para acceder al url que fue asignado.
Ejemplo:Link de Google

Tag: p

Paragraph se utiliza para determinar un conjunto de texto. Toma todo el ancho del contenedor del elemento.
Ejempo:

Hola Mundo!

Tag: h1, h2, ... h6

Heading se utiliza para determinar multiples niveles de titulos a lo largo del texto.
Ejemplo:

Esto es un titulo de nivel 3

Tag: div

Divisions es un contenedor con el proposito de dar formato al estilo. Usado generalmente con el proposito de contener grandes cantidades de texto.
Ejemplo:

Esto es un div

Tag: span

Spans Es otra manera de contener elementos para dar formato al estilo. Es un elemento en linea usado para identificar pequeños grupos de texto dentro de un elemento bloque.
Ejemplo:

Esto es texto Esto es span, regresamos a texto.

Tag: strong

Strong sirve para remarcar texto o enfasis.
Ejemplo: Esto es un strong: HOLAAA!!

Tag: i

Italic se usa para dar formato de italica a un texto
Ejemplo: Palabra normal -> Palabra italica

Tag: u

Underline se usa para dar formato de subrayado a un texto.
Ejemplo: Esto es una oracion subrayada.

Tag: b

Bold se utiliza para dar formato de negritas al texto.
Ejemplo: Palabra normal -> Palabra negritas

Tag: strike o del

Strike le da un formato como si el texto estuviera tachado o eliminado. Ahora en HTML 5 se utiliza el tag: del.
Ejemplo: Palabra normal -> Palabra tachada.

Tag: small

Small se utiliza para hacer el tamaño de texto mas pequeño del que html escribe actualmente
Ejemplo: Palabra normal -> Palabra small.

Tag: em

Emphasis se utiliza para dar emphasis a texto. Lo que hace el buscador es darle formato italica, pero este tag se debe de usar cuando se quiere dar claridad al texto.
Ejemplo:Palabra normal -> Palabra enfasis.

Tag: header

Header se utiliza generalmente al princio del decumento para contener una introducción e información de navegación y menus.
Ejemplo:

Esto esta dentro de un header

Tag: section

Section se utiliza con propositos estructurales de la pagina web. Ayuda a dar formato especial y dar organización al documento.
Ejemplo:

Esto es una seccion especial del documento

Tag: article

Article tiene el mismo proposito de section, pero de debe de usar todavía mas especificamente, no agrupando grandes secciones.
Ejemplo:

Esto es un artículo especial del documento

Tag: aside

Aside, a diferencia de section y article, el contenido debe de brindar información sobre la información principal
Ejemplo:

Esto es un aside especial del documento

Para dar información sobre la info principal

Tag: footer

Footer se utiliza para crear una sección de footer para dar información del autor, copyright, información de contacto u otra información relevante de la pagina web.
Ejemplo:

© 2023 A01783155_TC2005B. All rights reserved.

Tag: nav

Navigation es un tag que define una seccion de la pagina web donde se pueden encontrar links de navegacion dentro del mismo documetno para su facil movieminto
Ejemplo:

El más pupular de todos: Regresar al inicio
